# ZenOps

ZenOps 一站式运维平台，面向开发者和运维人员的统一数据库、服务器、对象存储等运维管理工具。

---

## 功能模块

### 已实现功能

- **用户角色权限控制（RBAC）**
  - 用户管理、角色管理、菜单权限管理
  - JWT 登录认证

- **数据源管理**
  - 多数据源配置与维护

- **SQL 执行窗口**
  - 在线 SQL 编辑与执行（Monaco Editor）
  - 多数据库适配（MySQL 等）
  - 多标签页 SQL 窗口
  - 执行结果表格渲染

- **SSH / SFTP 终端**
  - Web SSH 终端连接（xterm.js）
  - 多标签页会话管理
  - SFTP 文件浏览与操作
  - 服务器配置管理

- **对象存储管理**
  - 支持 MinIO、阿里云 OSS
  - Bucket / 文件管理

- **定时任务调度**
  - 基于 Quartz 的后端调度引擎（模块已就绪）
